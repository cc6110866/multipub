# MultiPub 用户手册

## 快速开始

### 什么是 MultiPub？

MultiPub 是一款 **Markdown 转公众号排版工具**。你只需用 Markdown 写好文章，选择喜欢的样式主题，一键复制即可粘贴到公众号编辑器。

### 3 步完成排版

```
1. 编辑 → 2. 选主题 → 3. 复制粘贴
```

---

## 功能介绍

### 1. Markdown 编辑器

左侧是 Markdown 编辑区，你可以：
- 直接输入 Markdown 文本
- 粘贴已有的 Markdown 内容
- 实时看到渲染效果

### 2. 实时预览

右侧是预览区，会实时显示：
- 文章最终效果
- 代码高亮效果
- 主题样式效果

### 3. 主题选择

顶部提供 5 套主题样式：

| 主题 | 风格 | 适用场景 |
|------|------|----------|
| 默认简洁 | 蓝色调，简洁大方 | 通用文章 |
| 科技蓝 | 深蓝配色，科技感 | 技术教程 |
| 暖色柔和 | 橙色调，温暖 | 生活情感 |
| 清新绿 | 绿色调，自然 | 环保健康 |
| 优雅紫 | 紫色调，设计感 | 设计艺术 |

### 4. 一键复制

点击「📋 复制到公众号」按钮，内容会自动复制到剪贴板，然后：
1. 打开公众号后台
2. 新建图文
3. 在编辑器中 Ctrl+V 粘贴
4. 样式完整保留！

## 支持的 Markdown 语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

### 段落

普通段落直接输入文字即可，段落之间用空行分隔。

### 强调

```markdown
**加粗文字**
*斜体文字*
`行内代码`
```

### 列表

```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

### 引用

```markdown
> 这是一段引用文字
> 可以多行
```

### 代码块

```markdown
\`\`\`javascript
function hello() {
  console.log('Hello World')
}
\`\`\`
```

支持的语言：JavaScript、TypeScript、Python、CSS、HTML、Bash、JSON、SQL、Java、Go、Rust、YAML

### 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
| D   | E   | F   |
```

### 链接

```markdown
[链接文字](https://example.com)
```

链接会自动在文末生成脚注，方便读者查看。

### 图片

```markdown
![图片描述](https://example.com/image.jpg)
```

### 分隔线

```markdown
---
```

## 使用技巧

### 技巧 1：代码高亮

技术文章建议选择「科技蓝」主题，代码块会显示更专业：

- 顶部有 Mac 风格的红黄绿按钮
- 语法高亮清晰
- 深色背景突出代码

### 技巧 2：表格排版

公众号表格样式有限，MultiPub 的表格：
- 自动添加边框
- 表头有背景色区分
- 宽度自适应

### 技巧 3：引用美化

引用块会自动添加：
- 左侧彩色边框
- 浅色背景
- 适当的内边距

### 技巧 4：字数统计

底部状态栏显示当前字数，方便控制文章长度。

## 常见问题

### Q1: 复制后粘贴到公众号，样式丢失了？

**可能原因**：
1. 使用了不支持的浏览器（建议使用 Chrome）
2. 公众号编辑器加载问题

**解决方法**：
1. 刷新公众号页面重试
2. 确保使用 Ctrl+V 粘贴（不是右键粘贴）
3. 检查浏览器版本是否过旧

### Q2: 代码块在公众号中显示异常？

**可能原因**：代码过长导致换行

**解决方法**：
1. 适当缩短代码行长度
2. 选择「科技蓝」主题，代码块样式更稳定

### Q3: 表格太宽怎么办？

**解决方法**：
1. 减少表格列数
2. 缩短单元格内容
3. 表格会自动横向滚动

### Q4: 图片无法显示？

**可能原因**：图片链接无法访问

**解决方法**：
1. 确保图片链接可公开访问
2. 建议先将图片上传到公众号图床
3. 使用公众号图片链接

### Q5: 支持哪些浏览器？

**推荐浏览器**：
- ✅ Chrome 90+
- ✅ Edge 90+
- ✅ Firefox 88+
- ✅ Safari 14+
